做市奖励观察

紫色区域是官方 liquidity rewards 市场的观察器。它只做评分和被动 bid/ask 建议，不会自动挂真实 maker 单。做市奖励更前置，但风险来自库存、价格跳变、成交后单边暴露和规则变化。

奖励市场 Top 候选

信号与订单

橙色区域表示跟单信号流。公开 API 延迟是目标钱包成交时间到本地第一次从 activity 接口看到该事件的时间，这正是跟单滞后的核心来源。
